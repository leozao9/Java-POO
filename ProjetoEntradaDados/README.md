# Projeto: ProjetoEntradaDados

Este projeto foi desenvolvido para praticar a **entrada de dados pelo teclado**  
em Java, utilizando a classe `Scanner`.

---

## 🎯 Objetivo

Permitir que o usuário insira informações via terminal, como nome, idade e outros dados,  
e exibir essas informações processadas na saída do console.

---

## 🧱 Estrutura das classes

- Classe principal que declara um objeto `Scanner` para capturar dados do teclado.
- Utiliza métodos do `Scanner` como `nextLine()`, `nextInt()`, etc.,  
  para leitura de diferentes tipos de entrada.

---

## 🛠️ Conceitos aplicados

- Entrada de dados com `Scanner`
- Tipos de dados primitivos
- Conversão e tratamento de entrada do usuário
- Exibição de resultados formatados
- Interação básica com o usuário

---

## 📌 Observações

Este projeto é essencial para entender como lidar com entradas externas em Java.  
Pode ser aprimorado com validação de dados, tratamento de exceções  
e criação de menus interativos para navegação.
